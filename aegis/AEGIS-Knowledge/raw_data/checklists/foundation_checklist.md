# Foundation Checklist

- [ ] Understand — restated the actual problem in your own words, not just the symptom
- [ ] Options — considered more than one approach
- [ ] Decision — picked using the priority order (Correctness → Security → Recoverability → Compatibility → Minimal Diff → Performance → Maintainability → Convention)
- [ ] Scope — explicit non-goals stated
- [ ] Verify — claims labeled as Verified / Reasoned / Assumed
- [ ] Communicate — risk and compatibility stated plainly in the report
